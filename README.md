# Customer Purchase & Revenue Analysis

## Project Overview
This project analyzes customer shopping behavior using Power BI.
The dataset contains transactions from shopping malls between 2021 and 2023.

## Tools Used
- Power BI
- Excel
- GitHub

## Key Analysis
- Shopping distribution by gender and age group
- Revenue analysis by gender and age
- Product category contribution
- Payment method preferences

## Key Insights
- Female customers generate higher revenue
- Adult and Middle-Age groups are core customers
- Clothing is the top revenue category
- Cash is the most used payment method
